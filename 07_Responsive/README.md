## Breakpoints más comunes

Celulares
* 320px el más chico
* 767px el más grande

Tablets
* 768px el más chico (iPad en posición vertical)
* 1280px el más grande (aprox.)

Pantallas escritorio
* 1360px para arriba (baja gama)
* 1600px para arriba (media / alta gama)
